# obfusimg Web (minimal)

This is a minimal browser port of the1 C++ project `obfusimg` by \_\_roselle\_\_ and pigeon.hannah.

## Features
- Gilbert-curve-based obfuscation (and inverse)
- Tent-map chaotic permutation obfuscation (and inverse)
- Runs fully in the browser (no backend)

## Run
- Open `index.html` in a modern browser.
- Alternatively, [an online version](https://obfusimg.netlify.app/) hosted by Netlify is deployed.
